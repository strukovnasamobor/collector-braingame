9. Curator                  = collector + opening book + MAST table + feature-weight learning
- policy:
    - Confiscator   - attackHeavy:  { placeOwn: 2, placeOpp:  6, placeDead: -2, elimOwn: -2, elimOpp: 6 }, inverted collector, chases opponent
    - Conservator   - defenseHeavy: { placeOwn: 6, placeOpp: -2, placeDead: -2, elimOwn: -2, elimOpp: 2 }, builds, flees opponent
    - Cumulator     - collectHeavy: { placeOwn: 6, placeOpp:  0, placeDead: -2, elimOwn: -2, elimOpp: 2 }, builds, ignores opponent
    - Collector     - heavy:        { placeOwn: 6, placeOpp:  2, placeDead: -2, elimOwn: -2, elimOpp: 4 }, builds, balanced offense/defense
    - Curator     - = Collector
- formula
    - PLACE phase:  weight = max(0.1, 1 + placeOwn·ownAdj  + placeOpp·oppAdj + placeDead·deadAdj)
    - ELIM phase:   weight = max(0.1, 1 + elimOwn·ownAdj   + elimOpp·oppAdj)
    - The features are simple counts of what's in the 8 neighbors of the candidate cell:
        - ownAdj - number of your own dots adjacent
        - oppAdj - number of opponent's dots adjacent
        - deadAdj - number of dead (eliminated) cells adjacent
